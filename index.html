Graphic Design

This image was intended to be applied to a shirt using screen-printing and was to be given to all of the students in that print course. The program’s logo is an eye, so I used the pen and width tools in Adobe Illustrator to draw it and create a hand-drawn look. I then wrote VCT and included the BGSU logo below.

Websites

This website was built for a local band, “Leadfood Granny”. I began this project by creating a persona- “Kai” likes to discover new artists and share them with his friends. Once deciding what information to include, I made a sitemap. I used this to plan out what information I should include on each page- this helped me when I began creating wireframes and comps. After presenting these to a client, I built the final site, the home page of that site is pictured above.

Layouts

This next design was made using Adobe Photoshop, InDesign, and Illustrator. It is a compilation of pictures taken during a Spring Break trip, as well as some illustrations I created using the pen and width tools. The Spotify code takes viewers to the song “Kokomo” by The Beach Boys on Spotify.

Photography

This image was taken on my Nikon D500. It was for an assignment on lighting. I was trying to capture the texture of her face while not making the image too light or dark. I love how rich her skin tone looks and the focus I was able to achieve.
